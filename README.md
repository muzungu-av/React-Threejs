# Приложение-пример реализация "физики" 3D объектов

Используется следующие библиотеки:

- ThreeJS - Собственно для отрисовки 3D объектов.


- react-three-fiber / react-three-drei - С этими библиотеками можно построить сцену декларативно, 
используя многократно используемые, самодостаточные компоненты, которые реагируют на состояние, 
легко взаимодействуют и могут участвовать в экосистеме React.


- react-three-cannon - Легкая трехмерная физика для Web (хук для fiber)

Результат работы этого кода можно посмотреть у меня на [Youtube](https://www.youtube.com/watch?v=g4UQDJyCnKg)
